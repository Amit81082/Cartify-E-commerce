📌 👉 ABOUT PROJECT
Cartify = Full Stack E-commerce Website

✔ Users can browse products
✔ Add / remove items from cart
✔ Secure checkout with payment
✔ Orders stored in database

👉 This type of system is standard in modern e-commerce apps
(product listing, cart, order, payment flow) ()

🛠 👉 INSTALLATION (STEP-BY-STEP)
STEP 1:
git clone https://github.com/Amit81082/Cartify-E-commerce

STEP 2:
cd Cartify-E-commerce

STEP 3:
npm install

STEP 4:
cd client && npm install

STEP 5:
npm run dev
🔐 👉 ENV SETUP
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
RAZORPAY_KEY=your_key
RAZORPAY_SECRET=your_secret
